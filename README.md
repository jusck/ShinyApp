# ShinyApp
Shiny App for Coursera Data Science Specialisation Assignment

This repository contains the code the following ShinyApp:

https://jusck.shinyapps.io/ShinyApp

It's purpose is to make predictions on Iris type based on a model.
The application has sliders to set lengths and widths of the petals and sepals.

A pitch for the application can be found at:

http://rpubs.com/jusck/IIS
